# KNOTEN

### AI-Powered Second Brain & Learning Workspace

KNOTEN is a modern AI-powered learning platform that helps users store knowledge, simplify difficult concepts, generate quizzes from notes, and discover personalized project ideas.

Inspired by the idea of connected knowledge systems, KNOTEN acts as a digital “second brain” where thoughts, notes, and learning resources become interactive and intelligent.

---

# ✨ Features

## 🧠 AI Knowledge Vault

* Save notes, ideas, and learning resources
* Upload text or documents
* AI-powered categorization and organization
* Semantic search across stored knowledge

---

## 🎓 Explain It Simply

Generate explanations in multiple styles:

* Like I’m 5
* Technical mode
* Teacher mode
* Story mode
* Real-life analogies

---

## 📚 Smart Quiz Generator

Turn notes into:

* MCQs
* Short-answer questions
* Flashcards
* Practice tests

Includes score tracking and interactive quizzes.

---

## 🧩 AI Project Idea Generator

Generate personalized project ideas based on:

* Skills
* Interests
* Experience level

Get:

* project roadmap
* recommended tech stack
* learning path
* difficulty estimation
